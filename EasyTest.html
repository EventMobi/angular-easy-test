<static> mockModule(moduleName, services)

Mocks a specified module that has already been registered with angular. Optionally registers a number of services with the mocked module.
Parameters:
Name Type Description
moduleName string The name of the module to mock.
services array An array of objects representing fake services to register with the module. Each object must have a name property representing the name of the service and a provider property which is the body of the service, i.e. an object that has the functions and properties of the service.
