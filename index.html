angular-easy-test

A framework that makes Angular Unit Test's easier to write.

install

npm install angular-easy-test --save-dev --save-exact

usage

Just include node_modules/lib/easy-test.js into your test files. Make sure it comes after you've included angular.js.

dependencies

Some parts of the library currently need Chai Expect to be defined as they run tests automatically. See EasyTest.testScope for an example of this.

See the jsdocs.
